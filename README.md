# DroneSwarm_simulator
A C++ simulation of autonomous drone swarm system behaviors. This project is a C++ Object-Oriented Programing application that simulate drone swarm moving in console. It demonstrates Separate Compilation by spliting classes into headers (.h) and source (.cpp) files

##How to build and run This project is configured for Visual Studio Code using the MinGW (g++) compiler.

Prerequisites
1)Visual Studio Code
2)MinGW/g++ compiler installed and added to system PATH.
Build Instructions
1)Open the project folder in VS Code.
2)Open any .cpp file (e.g., property_management_lab_1.cpp).
3)Press Ctrl + Shift + B to build the project.
4)Enter in terminal ".<filename>.exe"
